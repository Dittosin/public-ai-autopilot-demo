# 국민 AI 오토파일럿 데모

국민 AI 오토파일럿 정책제안서용 프론트엔드 데모입니다. 실제 정부, 금융, 민간 API와 연결하지 않고 `src/data/mockData.ts`의 mock data만 사용합니다.

권장 GitHub 저장소 설명:

> 국민 AI 오토파일럿 정책제안서용 프론트엔드 데모

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 터미널에 표시되는 로컬 주소를 열면 됩니다. 기본 개발 주소는 보통 `http://127.0.0.1:5173/`입니다.

## 빌드 확인

```bash
npm run build
```

## 정책제안서와의 연결

이 데모는 실제 API가 없는 프론트엔드 목업입니다. 정책제안서에서 제안하는 “전국민 AI 자율 대리인 / 국민 AI 오토파일럿”의 사용자 흐름을 시각화합니다.

시각화하는 핵심 흐름:

- 지원 확인
- 가능성 분류
- 목적별 동의
- 신청 준비
- 최종승인
- 감사로그

데모는 모두의 AI 기반 또는 기존 정부 AI 인프라와 연계해, 사용자의 생애사건을 기준으로 받을 수 있는 권리를 찾고 신청 가능한 상태까지 준비하는 경험을 보여줍니다. 공식 자격 판정과 처분은 소관기관 시스템이 수행한다는 전제를 화면에 명시합니다.

## 화면 흐름

- 온보딩: 앱의 목적과 사용자 승인 원칙 안내
- 홈: 받을 수 있는 지원 조회, AI 채팅 요청, 주요 패키지 표시
- 조회: 받을 수 있는 지원 목록과 가능성 분류 확인
- 목적별 동의: 조회 기관, 목적, 기간, 철회 가능 여부 확인
- 신청 준비: 신청서 초안과 첨부서류 상태 확인
- 승인: 제출 전 최종 확인과 본인인증 후 제출 버튼
- 기록: AI 조회 및 준비 이력, 대리권 철회
- 설정: 큰 글씨, 고대비, 쉬운 설명, 내 마이데이터 관리

## 코드 구조

- `src/components`: 공통 UI 컴포넌트
- `src/pages`: 주요 화면
- `src/data/mockData.ts`: 패키지, 동의 항목, 서류, 기록, 마이데이터 mock data
- `src/types.ts`: 화면과 데이터 타입
